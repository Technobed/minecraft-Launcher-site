直感的で美しいホーム画面

ランチャーのメイン画面では、Bedrock / Java の切り替えやテーマ変更がスムーズに行えます。 ガラスUIを採用し、視認性と美しさを両立しました。

アドオン管理がもっと簡単に

アドオンやテクスチャの追加をワンクリックで操作可能。 ZIP ファイルにも対応しており、複雑な解凍作業は不要です。 さらに、Java Edition のテクスチャ（ZIP形式）を Bedrock 統合版の形式へ自動変換する機能を搭載。 これにより、Java のリソースパックを統合版で手軽に利用できます。 追加・削除・切り替えも直感的に行えるため、初心者から上級者まで快適に利用できます。

Changelogs で更新内容をすぐ確認

最新バージョンの変更点を美しいカード形式で表示。 多言語対応で、世界中のユーザーにわかりやすく。

お好みのテーマカラーに

設定からテーマカラーを自由に変更可能。 ダークモードもサポートし、目に優しいデザインを実現しました。

設定画面を簡素化

設定画面を直感的に設計し、ユーザーが簡単に操作できるようにしました。 すべての設定項目を一目で確認できるよう、整理されています。

高度なアドオン管理機能

このランチャーは、アドオン管理をより便利にするための高度な機能を備えています。 ZIP ファイルをそのまま読み込み、必要なファイルを自動で抽出。 Java Edition のテクスチャを統合版用に変換する独自エンジンを搭載し、 複数アドオンの競合チェックや自動整理にも対応しています。
